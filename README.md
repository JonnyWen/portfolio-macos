# macOS-style Portfolio (React + Vite + Tailwind + GSAP)

This project is a macOS-inspired portfolio UI:
- Windowed UI (draggable windows)
- Dock with GSAP hover magnification
- Control Center theme switcher (Light / Dark / System)

## Quick start

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually http://localhost:5173).

## Customize
- Apps / windows: `src/constants/apps.js`
- Content: `src/apps/*`
- Theme store: `src/store/theme.js`
